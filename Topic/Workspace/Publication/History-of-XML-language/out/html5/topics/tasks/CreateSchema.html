How to create a schema language in Oxygen

An XML Schema describes the structure of an XML document. The XML Schema language is also referred to as XML Schema Definition (XSD). Here we describe the steps you need to take to create an XSD for your XML document.

  1. Open Oxygen XML editor
  2. Open your XML file e.g. File - Open - Browse to your .xml file and click Open
  3. Go to Tools>Generate/Convert Schema
